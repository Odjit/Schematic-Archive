# The Schematic Archive

The Schematic Archive is a community gallery for sharing V Rising castles and
other builds made for the [KindredSchematics](https://github.com/Odjit/KindredSchematics)
mod.

Each entry includes screenshots, build information, required DLC, and a
`.schematic` file that can be imported in-game.

## Submitting a Build

- Open a [**Submit Entry**](../../issues/new?template=submit-entry.yml) issue and
  provide the requested information.
- Accepted submissions will be added to the archive by the maintainers.

## Reporting an Entry

Open a [**Report Entry**](../../issues/new?template=report-entry.yml) issue if an
entry is broken, misleading, missing required DLC information, or otherwise needs
attention.
